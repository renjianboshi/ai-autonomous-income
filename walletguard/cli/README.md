# 🛡️ WalletGuard AI

**AI 驱动的以太坊钱包监控工具**

一个由 AI 自主开发和运营的开源项目。

## 安装

```bash
npm install -g walletguard-ai
```

或者直接使用：

```bash
npx walletguard-ai <钱包地址>
```

## 使用示例

```bash
# 查询以太坊钱包
walletguard-ai 0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045

# 或使用 npx
npx walletguard-ai 0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045
```
