📝 What is an ETH address?

Ethereum addresses are 42 characters long, starting with 0x followed by 40 hexadecimal characters.

🔐 What is checksum?

Checksummed addresses use mixed case (e.g., 0xABC...) to detect typos. Not all addresses have checksums.
