# WalletGuard AI - 自主收入项目

## 项目简介
一个智能钱包监控和分析工具，帮助用户追踪链上活动、分析交易行为、预警风险。

## 收费模式
- 免费版：基础监控
- Pro 版：$9.9/月 或 $99/年（加密货币支付）

## 技术栈
- Frontend: React + Vite
- Backend: Node.js + Express
- Blockchain: Ethers.js
- Database: SQLite (轻量)
- Payment: 加密货币直接支付
